## 入库结论

证据等级：`B`。Anthropic官方披露融资、估值、客户数和Claude Code收入，第三方估算收入增速极高。因未上市、收入净额口径和算力成本不透明，保留B级。

## 关键证据

1. 2026年2月Anthropic宣布300亿美元G轮融资，投后估值3800亿美元。
2. 官方披露超500家客户年化支出超100万美元，8家Fortune 10为Claude客户。
3. Claude Code 2025年5月开放，2026年披露年化收入超过25亿美元。
4. Anthropic宣布扩大Google Cloud TPU使用，2026年上线超过1GW容量。

## 人工复核点

1. 收入口径是gross还是net，云渠道分成如何影响利润。
2. AWS/Google算力交付与成本结构。
3. Claude Code增长可持续性。
4. 安全治理是否影响产品速度。
