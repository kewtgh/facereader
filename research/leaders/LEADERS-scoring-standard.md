# LEADERS评分标准与复核架构

版本：2026-05-01

本文件把《企业管理评分（内部决策版）》和《企业管理评分卡（Management Scorecard · 专业量表版）》转成可复用的评分口径。机器可读规则见 `assets/data/leaders-score-rubric.json`，自动审计脚本见 `assets/scripts/audit-leaders-score-rubric.mjs`。

## 1. 分数锚点

| 分数 | 含义 | 使用口径 |
|---:|---|---|
| 0-2 | 严重缺失 | 关键岗位或治理机制缺位，并已形成明显组织风险。 |
| 3-4 | 明显不足 | 有局部能力，但无法稳定支撑当前阶段业务。 |
| 5-6 | 中等可用 | 能维持基本运转，但存在一个或多个关键短板。 |
| 7-8 | 稳定有效 | 多数证据支持能力有效，仍需保留阶段、周期或披露折扣。 |
| 9-10 | 卓越系统化 | 必须证明能力可复制、可持续，不能由单次业绩或市场叙事直接推导。 |

## 2. 证据等级

| 等级 | 可接受证据 |
|---|---|
| A | 财报、公告、招股书、审计报告、正式任命、可复算经营数据。 |
| B | 多方公开报道、行业声誉、客户/产品案例、公司正式新闻稿。 |
| C | 单一访谈、媒体叙事、推断、营销材料或信息披露不足。 |

## 3. 七项维度口径

| 维度 | 核心问题 | 9分护栏 |
|---|---|---|
| 领袖气质 | 创始人是否能吸引强人才、承压并建立组织信任。 | 需要跨周期人才吸引、组织凝聚和外部资源号召的可验证证据。 |
| 决策力 | 创始人在战略方向、关键用人、资源配置和纠偏上是否持续高质量。 | 不能把收入、股价、融资或热度直接等同于决策力；必须看到关键取舍、关键用人和低代价纠偏。 |
| 实干性 | 组织是否把方向转化为产品、交付、利润、现金流或客户复购。 | 需要重复兑现的结果证据，单次爆发或行业beta只能支撑7-8分。 |
| 补位力 | 二三号位是否补齐创始人短板。 | 需要多名关键高管、跨职能协同和继任/授权线索。 |
| 文化契合度 | 文化是否匹配业务阶段、客户价值、成本纪律和长期目标。 | 需要文化与经营结果互相印证，不能只看价值观表述。 |
| 岗位专长完整度 | 关键职能是否有足够专业的人和机制覆盖。 | 研发、产品、销售、财务、法务合规、人力、供应链或行业关键职能需均有强人和机制。 |
| 专业化治理结构 | 董事会、财务披露、内控、授权和风险管理是否足以约束组织。 | 需要上市公司级强披露或等价治理证据；治理争议、非上市不透明或资本依赖应降分。 |

## 4. 自动复核规则

1. 任何9分都必须有 `sources` 或 `research_file`，且公司证据等级不能是C。
2. C级证据下，创始人决策力最高8分。
3. 治理结构9分必须有A级强披露；B级证据治理最高8分，C级最高6分。
4. C级证据下，岗位专长完整度最高7分。
5. LEADERS均分高于Darwin均分0.5分以上进入半年度观察；高于1.2分以上进入人工复核清单，但不阻断构建。

## 5. 本轮校准结论

本轮复核排除寒武纪，重点下调了证据门槛不足的9分：DeepSeek、OpenAI、Anthropic、宇树科技、字节跳动、威马汽车、汇顶科技、韦尔股份、闻泰科技、平头哥、涂鸦智能、依图科技、领创集团等。宁德时代保留实干性和岗位完整度9分，因为其A级公开材料能支撑龙头组织系统化，但决策力和治理结构下调到8分，避免用行业地位替代创始人决策与治理质量。
